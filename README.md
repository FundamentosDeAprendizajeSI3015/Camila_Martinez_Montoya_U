# Máquinas Automatizadas

Este repositorio contiene el desarrollo de diferentes implementaciones, pruebas y análisis relacionados con máquinas automatizadas. A lo largo del proyecto se trabajan distintos enfoques, incluyendo procesamiento de datos, modelos y experimentos organizados por entregas y módulos.

---

## Objetivo

Desarrollar e implementar soluciones relacionadas con máquinas automatizadas, explorando diferentes técnicas y herramientas para analizar, modelar y resolver problemas de manera estructurada.

---

## Tecnologías utilizadas

- Python  
- Jupyter Notebook  
- Librerías estándar de análisis y modelado  

---

## Estructura del proyecto

El repositorio está organizado por entregas, informes y módulos de trabajo.
